# IOT first tasks for CO-OP training in SMart methods 

## The 1st task is crate a simple Blink code for running the ESP32

1. First you must download the Arduino IDE.
![image](https://user-images.githubusercontent.com/109042265/178161288-a08041dc-1b5b-40b4-825c-c3608c47e72e.png)

2. After you download the program you must add Additional Board Mangers form the Preferences menu.
![image](https://user-images.githubusercontent.com/109042265/178161348-cc068594-3087-4c70-b6f8-9b6b24e26f88.png)

3. Then you download the esp32 package form the board manger located at the tools drop-down menu.
![image](https://user-images.githubusercontent.com/109042265/178161379-40b9d82d-e990-4105-995e-4f62b4dd2dcf.png)

4. You shoud also choose the port with the connected board.

![image](https://user-images.githubusercontent.com/109042265/178161418-ff2aa00d-33f2-49e9-b3aa-f1f82d3e63da.png)

5. After finishing the above you can star writing the code that you want to upload to the board you have.
